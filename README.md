International Green Power Utilities Financial Analysis Tool

1. Problem & User
This project provides an interactive financial analysis tool for global green power utilities, enabling users to explore, compare, and evaluate the financial performance of leading international renewable energy companies. It supports investors, students, and researchers to analyze profitability, stability, and growth efficiently.

2. Data
Source: WRDS Compustat Global Fundamentals Annual
Access Date: April 2026
Key Fields: Revenue, Net Income, Operating Income, Total Assets, Total Liabilities, Equity, ROE, Debt Ratio, Profit Margins, Growth Rates

3. Methods
Load and clean CSV data with automated column standardization
Calculate financial ratios: ROE, net margin, gross margin, debt ratio, current ratio
Compute growth metrics for revenue, net income, and total assets
Build interactive multi-tab dashboard using Streamlit
Visualize trends and comparisons with Plotly charts
Support company selection, year filtering, and data export

4. Key Findings
The tool enables year‑by‑year trend analysis for revenue, profitability, and leverage
Cross-company comparison identifies leaders in profitability and financial stability
Visual dashboards highlight growth patterns and risk levels across green power firms
Financial ratios support consistent evaluation of operational efficiency
Export function allows offline analysis and reporting
